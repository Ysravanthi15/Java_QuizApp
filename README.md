## 📌 Project Title: Quiz Application (Java Swing)

### 📖 Description

The **Quiz Application** is a desktop-based software developed using Core Java and Java Swing that allows users to attempt multiple-choice questions in an interactive environment. The application provides a simple and intuitive graphical user interface where users can select answers, navigate through questions, and receive a final score at the end of the quiz.

The system is designed using a modular approach by separating data, logic, and user interface, ensuring better maintainability and scalability.

---

### 🎯 Objectives

* To develop an interactive quiz system using Java GUI
* To implement object-oriented programming concepts
* To provide timed assessments with automatic evaluation
* To create a user-friendly interface for better user experience

---

### ⚙️ Features

* ✔ Multiple-choice questions (MCQs)
* ✔ Timer-based question navigation
* ✔ Automatic answer validation
* ✔ Score calculation and display
* ✔ Simple and responsive GUI
* ✔ Restart option for repeated attempts

---

### 🛠️ Technologies Used

* **Programming Language:** Java
* **GUI Framework:** Java Swing
* **Concepts:** OOP (Encapsulation, Modularity), Event Handling

---

### 🧩 System Design

The project follows a structured architecture:

* **Question Class:** Stores quiz data (questions, options, answers)
* **QuizService Class:** Handles business logic (score, validation, flow)
* **QuizFrame Class:** Manages GUI and user interaction
* **Main Class:** Entry point of the application

---

### ▶️ Working

1. The application starts from the main class.
2. The GUI displays one question at a time with four options.
3. The user selects an answer and clicks “Next” (or auto-moves via timer).
4. The system validates the answer and updates the score.
5. After all questions are completed, the final score is displayed.
